# meadowhop

MeadowHop is a cozy top-down GODOT game where your goal is to clear the map as quickly as possible, while your time is tracked.

HOW TO PLAY
1. Open itch.io (https://rithunyaa.itch.io/meadowhop)
2. Click full-screen (works best this way)
3. Click Play
4. Click anywhere on the screen to start the game once you have read the instructions
5. Left-click once to destroy the trees and rock
   -They will drop logs and stones that you do not HAVE to pick up
6. Once you clear the island fully, a pop-up will show how long it took you
